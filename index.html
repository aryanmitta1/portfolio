Rocket Payload Telemetry System

Payload Electronics Lead, live downlink to NASA

NYU ROGUE AEROSPACE · 144–148 MHz VHF

Led payload electronics for a NASA University Student Launch Initiative rocket, designing the full telemetry chain that transmitted live flight data to NASA's ground receiver during flight.   • Planned battery, shielding & manual override; authored NASA review docs.

AEROSPACE · ML / CV

Celestial Navigation System

Position fixing from the sky & skyline

PYTORCH · OPENCV · ASTROMETRY

A vision and ML system that estimates geographic position from celestial and skyline imagery, with no GPS required.

  • Generated a large training set of 100k sky images in Stellarium, spanning many locations and times.
  • Trained a 1D convolutional neural network on the identified star data to regress a position on Earth.
  • Used Haversine loss so the network optimizes for real-world great-circle distance rather than raw coordinate error.
  • After star identification, computed the final fix in MATLAB using spherical trigonometry.
  • Reached fixes accurate to roughly a mile.
